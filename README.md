# APT-Project
How to Run (VS Code)

1.Open the project folder in VS Code
2.Open Main.java
3.Click the "Run" button


How to Run (Terminal)

1.Open terminal in project folder
2.Navigate to src:
cd src
3.Compile all files:
javac Main.java crdt/character/.java crdt/block/.java utils/*.java
4.Run the program:
java Main
